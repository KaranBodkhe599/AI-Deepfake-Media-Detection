Deepfake Media Detection System

An AI-powered system designed to detect fake news and manipulated media using advanced analysis techniques.

Project Overview

This project focuses on identifying deepfake images and misleading news content by combining Natural Language Processing and Computer Vision. It analyzes both text and images from news articles and generates a final authenticity score along with explanation.

Objective

The main goal is to build a reliable and intelligent system that can automatically verify the authenticity of online information and help users detect misinformation quickly and efficiently.

Key Features

  • Real-time news article scanning
  • AI-based text analysis
  • Image authenticity detection
  • Final trust score with verdict
  • Explainable AI output (analysis + confidence)
  • Scan history storage
